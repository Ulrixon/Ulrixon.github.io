# Ulrixon.github.io

This repository contains the personal academic/professional website and application materials for Cheng-Shun (Ryan) Chuang.

## Purpose
- **Personal website** for portfolio, research, and professional profile, hosted at [ulrixon.github.io](https://ulrixon.github.io)
- **PhD/graduate school application support**: includes a LaTeX-generated, up-to-date academic CV/resume, publications, and project highlights
- **Showcase** of research, technical skills, and selected GitHub repositories

## Structure & Recent Changes
- **Multi-page website**: `index.html` (summary/landing), `projects.html` (projects & GitHub repos), `publications.html` (research papers)
- **Shared stylesheet**: `style.css` for consistent, modern design
- **Resume/CV**: `resume.tex` (LaTeX source) and `resume.pdf` (compiled, always 1-page, PhD-optimized)
    - Coursework and experience updated from transcript and work history
    - Research interests, publications, and projects highlighted
    - All links point to the new website and subpages
- **.gitignore**: ignores LaTeX build artifacts

## How to Use
- View the live site at: [https://ulrixon.github.io](https://ulrixon.github.io)
- Edit `resume.tex` and recompile with `pdflatex resume.tex` to update the PDF
- Update HTML or CSS as needed for new content or design

## Main Files
- `index.html` — Landing page (summary, highlights, about, education, experience, skills, contact)
- `projects.html` — Projects and GitHub repositories
- `publications.html` — Research publications
- `resume.tex` / `resume.pdf` — LaTeX source and compiled academic CV
- `style.css` — Shared stylesheet
- `.gitignore` — Ignores LaTeX intermediate files

This repo is intended as a professional, academic, and technical showcase for graduate applications and networking.
