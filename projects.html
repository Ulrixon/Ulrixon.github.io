Virginia Tech Transportation Safety Index (VTTSI) Sep 2025 – Dec 2025
FastAPI Streamlit GCP Cloud Run Cloud SQL Empirical Bayes MCDM Python
  • Built a real-time safety scoring system generating 0–100 scores every 15 minutes by fusing VDOT crash history with connected-vehicle telemetry.
  • Developed a FastAPI backend and Streamlit frontend, deployed on Google Cloud Run with Cloud SQL for persistent data storage.
  • Implemented Empirical Bayes crash modeling, uplift factors (speed deficit, variance, VRU conflicts), and a CRITIC-weighted hybrid MCDM framework (SAW–EDAS–CODAS) for corridor ranking.
  • Integrated Cloud Scheduler for automated periodic scoring and a monitoring dashboard showing historical trends.
